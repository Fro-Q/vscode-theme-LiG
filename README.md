## Match with System

<!--eslint-skip-->

```jsonc
// .vscode/setting.json
{
  "window.autoDetectColorScheme": true,
  "workbench.preferredLightColorTheme": "LiG Light",
  "workbench.preferredDarkColorTheme": "LiG Dark",
}
```
